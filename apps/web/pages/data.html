Data & Privacy

A plain explanation of what ClearPilot stores, where, and how it moves between the web app and the desktop app.

What's stored

  • Your account: email, display name, and a securely hashed password (never the password itself)
  • Materials you upload: resumes, job descriptions, project notes, training Q&A
  • Your knowledge base entries and interview/study history

Where it lives

Everything above is stored in a Postgres database we run on Railway, reachable only over a private network from the ClearPilot backend — never exposed directly to the internet, and never bundled into the desktop app itself.

How web and desktop stay in sync

One account, one backend. The desktop app keeps a local copy of your materials and history so it keeps working instantly and offline during an actual interview, and syncs with the same account in the background — it never talks to the database directly.

Third parties

Questions you ask are sent to OpenAI's API to generate answers. Resume and job description text you upload may be included in that request as context. No material is sold or shared for advertising.
