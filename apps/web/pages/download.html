How to get started

  1. 1

    Create a ClearPilot account

    Sign up at clearpilot.shop — you'll use the same login in the app.

  2. 2

    Install & sign in

    Run the .exe installer, open ClearPilot, and log in with your account.

  3. 3

    Add your OpenAI API key

    Settings → OpenAI tab. The key is stored locally and never leaves your machine.

  4. 4

    Start your interview

    Pick an interview from your dashboard, hit Speaker or Mic, and ClearPilot listens live.

Windows requirements

  • Windows 10 or 11 (64-bit)
  • 4 GB RAM minimum
  • Internet connection for AI features
  • OpenAI API key with Realtime API access
